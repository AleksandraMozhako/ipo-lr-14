
## **Задание 1**

Создать два связанных класса для обработки изображений:

### **Класс `ImageHandler`**
Класс для базовой работы с изображениями:
- Принимает путь к изображению при инициализации.
- Содержит методы:
  - Загрузка изображения.
  - Сохранение изображения.
  - Изменение размера изображения.
  - Передача изображения в класс `ImageProcessor`.

---

### **Класс `ImageProcessor`**
Класс для обработки изображений:
- Принимает изображение, переданное из `ImageHandler`.
- Содержит методы:
  - Применение фильтров.
  - Добавление текста.
  - Поворот изображения.

---

## **Задание 2 (Вариант 4)**

### **Требования к классу `ImageHandler`**
1. Реализовать метод для изменения формата изображения на **JPG**.
2. Реализовать метод для поворота изображения на **45 градусов**.

---

### **Требования к классу `ImageProcessor`**
1. Применить фильтр повышения резкости (**SHARPEN**).
2. Добавить рамку вокруг изображения с шириной **15 пикселей**.
